Cliente Externo

Página HTML + JavaScript que consume la API REST de Cine La Estación.

Tipo de cambio

Consulta pública: GET /api/consultas/tipo-cambio

Cartelera

Consulta pública: GET /api/peliculas

Acceso administrativo

Autenticación HTTP Basic contra la API de Spring Boot.

Mantenimiento de snacks

Requiere rol ADMIN: POST, PUT y DELETE /api/snacks

Reservas

Requiere rol ADMIN o CAJERO: GET /api/reservas
